# House Price Prediction with CML

This is a hands-on machine learning project built to explore **Continuous Machine Learning (CML)** workflows by automating model training, evaluation, and reporting inside GitHub Actions.

Using a **Random Forest Regressor**, the project predicts house prices based on property features, generates performance metrics, and automatically posts model reports with feature importance visualizations on pull requests using **CML**.

### Highlights

* Automated ML workflow with **GitHub Actions + CML**
* Model evaluation with **Train/Test MAE**
* Automated feature importance reporting on PRs
* Demonstrates practical **MLOps + CI/CD for ML**

This project was created as a learning initiative to understand how **machine learning pipelines can be integrated into modern DevOps workflows**, with a focus on reproducibility, automation, and scalable experimentation.
Evaluated performance using Train/Test MAE and improved results through feature engineering.
Integrated CML with **GitHub Actions to automate training, generate reports, and post model insights directly on pull requests**.
